INTRODUCTION
------------
Paint like.


DESCRIPTION
------------
-> Drawing Functions (pencil, brush, eraser, colors)

-> Graphical Interface (Gui GTK) with a drawing area

-> Binary Handler (Browse, Open, Save files)

-> Use of some filters (Black&White, Invert, Flip, Rotate)

++ Handle Ctrl-Z


INSTALLATION
------------
To install :

   $ make
   
   $ ./Tinypaint
